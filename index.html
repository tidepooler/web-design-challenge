Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes approaching the equator. To accomplish this analysis, data was obtained from the OpenWeatherMap API. After populating a dataset using these data on over 500 cities, Matplotlib was used to plot various aspects of weather compared to latitude. Factors included temperature, cloudiness, wind speed, and humidity. The analysis took place on January 17th, 2017.

This site provides source data and visualizations for the analysis. It also provides the data table obtained, and visualizations created as part of the analysis. Any trends depicted with correlations are explained and provided.
